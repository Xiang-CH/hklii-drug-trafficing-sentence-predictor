# Sample Judgment types for Feature Extraction

- `d` stands for Defendant
- `dt` stands for Drug Trafficking
- `ndt` stands for Non-Drug Trafficking
## File to Judgment Type Mapping

| File Name | Judgment Type |
|-----------|---------------|
| `single-d-single-dt.htm` | Single Defendant Facing Single DT Charge |
| `single-d-multi-dt.htm` | Single Defendant Facing Multiple DT Charges |
| `single-d-dt+ndt.htm` | Single Defendant Facing DT Charges and Non-DT Charges |
| `multi-d-single-dt.htm` | Multiple Defendants Facing Single DT Charge Together |
| `multi-d-multi-dt.htm` | Multiple Defendants Facing Multiple DT Charges |
| `multi-d-multi-dt+ndt.htm` | Multiple Defendants Facing DT Charges and Non-DT Charges |
| `corridendum.htm` | Judgments with Corrigendum |
| `case-with-corridendum.htm` | Judgments with Corrigendum |
| `appeal.htm` | Cases with Appellate Judgments |
| `appeal-from.htm` | Cases with Appellate Judgments |
